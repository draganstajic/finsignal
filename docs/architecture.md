# FinSignal Architecture

## Purpose
FinSignal is an event-driven midlayer designed to simulate fintech invoice workflows with full correlation and auditability.

## Core principles
- Signals over requests  
- Correlation-first design  
- Audit as a first-class citizen  
- AI-ready observability  

## Phase 0.1 scope
- Invoice ingestion  
- Event emission  
- Audit trail  
- Workflow visibility
